ФУНКЦИЯ 1: Поиск услуг
----------------------

№1
Название: Поиск по полному названию
Функция: Поиск
Шаги: 1. Войти в приложение. 2. Ввести в поле поиска «Худи «Акацуки Манга»». 3. Нажать Enter.
Ожидаемый результат: Отображается одна услуга с названием «Худи «Акацуки Манга»»
Фактический результат: Отображается одна услуга
Комментарии: Пройден

№2
Название: Поиск по частичному совпадению
Функция: Поиск
Шаги: 1. Войти. 2. Ввести в поиск «Футболка». 3. Нажать Enter.
Ожидаемый результат: Отображаются все услуги, содержащие слово «Футболка»
Фактический результат: Отображаются все футболки
Комментарии: Пройден

№3
Название: Поиск без результатов
Функция: Поиск
Шаги: 1. Войти. 2. Ввести в поиск «Несуществующая услуга». 3. Нажать Enter.
Ожидаемый результат: Список пуст, индикатор показывает «0-0 из 0»
Фактический результат: Список пуст, индикатор «0-0 из 0»
Комментарии: Пройден

№4
Название: Поиск с учетом регистра
Функция: Поиск
Шаги: 1. Войти. 2. Ввести в поиск «худи» (строчными буквами).
Ожидаемый результат: Отображаются все худи независимо от регистра
Фактический результат: Отображаются все худи
Комментарии: Пройден

№5
Название: Очистка поиска
Функция: Поиск
Шаги: 1. Войти. 2. Ввести в поиск «Худи». 3. Очистить поле поиска.
Ожидаемый результат: Отображаются все услуги текущей категории
