## 2. Activity диаграмма (Диаграмма активности)

### 2.1 Процесс авторизации

```mermaid
flowchart TD
    Start([Начало]) --> Input[Ввод логина и пароля]
    Input --> Validate{Поля заполнены?}
    Validate -->|Нет| ShowError1[Показать ошибку: Введите логин и пароль]
    ShowError1 --> Input
    Validate -->|Да| ConnectDB[Подключение к БД]
    ConnectDB --> QueryUser[Запрос пользователя из БД]
    QueryUser --> CheckUser{Пользователь найден?}
    CheckUser -->|Нет| ShowError2[Показать ошибку: Неверный логин или пароль]
    ShowError2 --> Input
    CheckUser -->|Да| LoadUserData[Загрузка данных пользователя и роли]
    LoadUserData --> OpenMain[Открыть главное окно]
    OpenMain --> CloseLogin[Закрыть окно авторизации]
    CloseLogin --> End([Конец])
```

### 2.2 Процесс управления услугами

```mermaid
flowchart TD
    Start([Начало]) --> CheckRole{Роль пользователя?}
    CheckRole -->|Пользователь| ViewOnly[Только просмотр услуг]
    CheckRole -->|Модератор/Админ| ShowAddBtn[Показать кнопку Добавить услугу]
    
    ViewOnly --> Browse[Просмотр списка услуг]
    ShowAddBtn --> Browse
    
    Browse --> Action{Действие пользователя}
    Action -->|Поиск| Search[Ввод текста поиска]
    Search --> Filter[Фильтрация списка]
    Filter --> Browse
    
    Action -->|Фильтр коллекции| SelectCollection[Выбор коллекции]
    SelectCollection --> Filter
    
    Action -->|Смена категории| ChangeTab[Переключение вкладки Кастом/Косплей]
    ChangeTab --> Filter
    
    Action -->|Навигация| ChangePage[Переход на другую страницу]
    ChangePage --> Browse
    
    Action -->|Добавить услугу| CheckRoleAdd{Есть права?}
    CheckRoleAdd -->|Нет| Browse
    CheckRoleAdd -->|Да| OpenAddDialog[Открыть окно добавления]
    OpenAddDialog --> FillForm[Заполнить форму услуги]
    FillForm --> UploadImage{Загрузить фото?}
    UploadImage -->|Да| SelectImage[Выбрать файл изображения]
    SelectImage --> ValidateForm
    UploadImage -->|Нет| ValidateForm{Форма валидна?}
    ValidateForm -->|Нет| ShowFormError[Показать ошибку валидации]
    ShowFormError --> FillForm
    ValidateForm -->|Да| CopyImage[Копировать изображение в Resources]
    CopyImage --> SaveToDB[Сохранить услугу в БД]
    SaveToDB --> CloseDialog[Закрыть диалог]
    CloseDialog --> ReloadServices[Перезагрузить список услуг]
    ReloadServices --> Browse
    
    Action -->|Редактировать| CheckRoleEdit{Есть права?}
    CheckRoleEdit -->|Нет| Browse
    CheckRoleEdit -->|Да| OpenEditDialog[Открыть окно редактирования]
    OpenEditDialog --> LoadServiceData[Загрузить данные услуги]
    LoadServiceData --> FillForm
    
    Action -->|Выход| Logout[Выход из системы]
    Logout --> End([Конец])
```

## 3. DFD диаграмма (Диаграмма потоков данных)

```mermaid

    System -->|Данные пользователя, список услуг| User
```

## Описание диаграмм

### Use Case диаграмма
Показывает взаимодействие различных ролей пользователей (Гость, Пользователь, Модератор, Администратор) с функциями системы. Модераторы и администраторы имеют расширенные права на управление услугами.

### Activity диаграммы
1. **Процесс авторизации**: Детально описывает шаги входа в систему с валидацией данных
2. **Процесс управления услугами**: Показывает полный цикл работы с услугами, включая просмотр, поиск, фильтрацию, добавление и редактирование

### DFD диаграммы
1. **Уровень 0**: Контекстная диаграмма показывает систему как единое целое с внешними сущностями
2. **Уровень 1**: Детализирует основные процессы системы и потоки данных между ними

### Диаграмма классов
Показывает структуру классов приложения, их атрибуты, методы и взаимосвязи. Включает модели данных (User, Service, Collection), окна (LoginWindow, MainWindow, EditServiceWindow) и вспомогательные классы (Database, Program, App).
