Руководство пользователя системы Matieu
========================================

1. ВВЕДЕНИЕ

Система Matieu — это приложение для управления каталогом услуг по пошиву кастомной одежды и косплей-костюмов. Приложение предоставляет удобный интерфейс для просмотра, поиска и управления услугами, разделенными по категориям и коллекциям.

Система поддерживает четыре роли пользователей:
- Администратор (admin) — полный доступ ко всем функциям
- Модератор (moderator) — управление услугами
- Мастер (master) — просмотр услуг
- Пользователь (user) — просмотр услуг

2. ЗАПУСК ПРИЛОЖЕНИЯ И АВТОРИЗАЦИЯ

Запуск приложения осуществляется с окна авторизации, где пользователь должен ввести логин и пароль для получения доступа к системе (Рисунок 1).

[Рисунок 1 — Окно авторизации]

Для входа в систему:
1. Введите логин в поле "Логин"
2. Введите пароль в поле "Пароль"
3. Нажмите кнопку "Войти"

При успешной авторизации откроется главное окно приложения с доступным функционалом в зависимости от роли пользователя.

Возможные ошибки при авторизации:
- "Введите логин и пароль" — одно или оба поля не заполнены
- "Неверный логин или пароль" — учетные данные не найдены в системе

Тестовые учетные записи:
- Администратор: логин "admin", пароль "admin123"
- Модератор: логин "moder1", пароль "moder123"
- Пользователь: логин "ivan_petrov", пароль "pass123"

3. ГЛАВНОЕ ОКНО ПРИЛОЖЕНИЯ

После успешной авторизации открывается главное окно системы, которое содержит следующие элементы интерфейса (Рисунок 2):

[Рисунок 2 — Главное окно приложения]

Верхняя панель:
- Кнопка "Услуги" — обновление списка услуг
- Кнопка "Выход" — выход из системы и возврат к окну авторизации
- Кнопки управления окном (свернуть, закрыть)

Панель фильтрации:
- Поле поиска — поиск услуг по названию
- Выпадающий список "Коллекция" — фильтрация по коллекциям
- Вкладки "Кастом" и "Косплей" — переключение между категориями услуг

Центральная область:
- Список услуг с карточками (отображается по 3 услуги на странице)
- Кнопки навигации "Назад" и "Вперед" для перехода между страницами
- Информация о текущей странице (например, "1-3 из 15")

Кнопка "Добавить услугу" (видна только для модераторов и администраторов)

4. ПРОСМОТР УСЛУГ

4.1 Структура карточки услуги

Каждая услуга отображается в виде карточки, содержащей:
- Фотография услуги (120x120 пикселей)
- Название услуги
- Название коллекции (если услуга входит в коллекцию)
- Цена в рублях
- Описание услуги (если указано)
- Кнопка "Редактировать" (только для модераторов и администраторов)

4.2 Переключение между категориями

Система поддерживает две категории услуг:
- Кастом — кастомная одежда с уникальными принтами
- Косплей — костюмы для косплея персонажей

Для переключения между категориями используйте вкладки в верхней части окна. При переключении категории список услуг автоматически обновляется, и счетчик страниц сбрасывается на первую страницу.

4.3 Поиск услуг

Для поиска услуги по названию:
1. Введите текст в поле поиска в верхней части окна
2. Список услуг автоматически отфильтруется по введенному тексту
3. Поиск не чувствителен к регистру

Пример: введите "худи" для поиска всех худи в текущей категории.

4.4 Фильтрация по коллекциям

Система поддерживает следующие коллекции:
- Аниме
- Новый год
- Хэллоуин
- Киберпанк
- Нуар

Для фильтрации услуг по коллекции:
1. Откройте выпадающий список "Коллекция"
2. Выберите нужную коллекцию
3. Список услуг автоматически отфильтруется

Для отображения всех услуг выберите "Все коллекции".

4.5 Навигация по страницам

Услуги отображаются постранично по 3 карточки на странице. Для навигации используйте:
- Кнопка "Назад" — переход на предыдущую страницу
- Кнопка "Вперед" — переход на следующую страницу
- Текстовая информация показывает текущий диапазон и общее количество услуг

Кнопки навигации автоматически становятся неактивными при достижении первой или последней страницы.

5. УПРАВЛЕНИЕ УСЛУГАМИ (для модераторов и администраторов)

5.1 Добавление новой услуги

Для добавления новой услуги:
1. Нажмите кнопку "Добавить услугу" в главном окне
2. Откроется окно "Добавление услуги" (Рисунок 3)
3. Заполните обязательные поля:
   - Название услуги
   - Цена (в рублях, можно использовать точку или запятую для десятичных дробей)
4. Заполните дополнительные поля (по желанию):
   - Описание услуги
5. Выберите категорию из выпадающего списка (Custom или Cosplay)
6. Выберите коллекцию из выпадающего списка (или оставьте "Без коллекции")
7. Нажмите кнопку "Выбрать фото" для загрузки изображения услуги
8. Нажмите кнопку "Сохранить"

[Рисунок 3 — Окно добавления услуги]

При загрузке фотографии:
- Поддерживаются форматы: JPG, JPEG, PNG
- Изображение автоматически копируется в папку Resources приложения
- Если файл с таким именем уже существует, к имени добавляется временная метка

Возможные ошибки:
- "Заполните название и цену" — не заполнены обязательные поля
- "Неверный формат цены" — введено некорректное значение цены

5.2 Редактирование услуги

Для редактирования существующей услуги:
1. Найдите нужную услугу в списке
2. Нажмите кнопку "Редактировать" на карточке услуги
3. Откроется окно "Редактирование услуги" с заполненными данными (Рисунок 4)
4. Измените необходимые поля
5. При необходимости загрузите новое фото (старое будет заменено)
6. Нажмите кнопку "Сохранить"

[Рисунок 4 — Окно редактирования услуги]

При сохранении изменений:
- Автоматически обновляется поле "Дата обновления" (updated_at)
- Список услуг в главном окне автоматически обновляется
- Если фото не было изменено, сохраняется ссылка на старое изображение

Для отмены изменений нажмите кнопку "Отмена" или закройте окно.

6. ВЫХОД ИЗ СИСТЕМЫ

Для выхода из системы:
1. Нажмите кнопку "Выход" в верхней панели главного окна
2. Откроется окно авторизации
3. Текущая сессия будет завершена

Для закрытия приложения:
1. Нажмите кнопку закрытия окна (X) в правом верхнем углу
2. Появится диалоговое окно подтверждения "Закрыть приложение?"
3. Нажмите "Да" для закрытия или "Нет" для отмены

7. ОСОБЕННОСТИ РАБОТЫ С СИСТЕМОЙ

7.1 Автоматическое обновление данных

- При добавлении или редактировании услуги список автоматически обновляется
- Фильтры и поиск сохраняются при обновлении списка
- Текущая страница сбрасывается на первую при изменении фильтров

7.2 Работа с изображениями

- Изображения хранятся в папке Resources/pr/Custom или Resources/pr/Cosplay
- При отсутствии изображения отображается заглушка "Фото"
- Изображения масштабируются с сохранением пропорций

7.3 Разграничение прав доступа

Роль "Пользователь" и "Мастер":
- Просмотр списка услуг
- Поиск и фильтрация услуг
- Навигация по страницам
- Переключение между категориями

Роль "Модератор" и "Администратор":
- Все функции пользователя
- Добавление новых услуг
- Редактирование существующих услуг
- Загрузка и изменение изображений

8. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

Для работы приложения необходимо:
- Операционная система: Windows, Linux или macOS
- .NET 8.0 или выше
- PostgreSQL база данных с настроенной схемой
- Минимальное разрешение экрана: 1024x768

Подключение к базе данных настраивается в классе Database (файл Database.cs).

9. РЕШЕНИЕ ТИПИЧНЫХ ПРОБЛЕМ

Проблема: Не отображаются изображения услуг
Решение: Убедитесь, что файлы изображений находятся в папке Resources/pr/Custom или Resources/pr/Cosplay

Проблема: Ошибка подключения к базе данных
Решение: Проверьте настройки подключения в файле Database.cs и убедитесь, что PostgreSQL запущен

Проблема: Не сохраняется новая услуга
Решение: Проверьте, что заполнены все обязательные поля (название и цена) и формат цены корректен

Проблема: Кнопка "Добавить услугу" не отображается
Решение: Эта кнопка доступна только для пользователей с ролью "moderator" или "admin"

10. ЗАКЛЮЧЕНИЕ

Система Matieu предоставляет удобный интерфейс для управления каталогом услуг по пошиву одежды. Интуитивно понятный интерфейс, мощные возможности поиска и фильтрации, а также разграничение прав доступа делают систему эффективным инструментом для работы с каталогом услуг.

Для получения дополнительной информации обратитесь к технической документации проекта или к администратору системы.
